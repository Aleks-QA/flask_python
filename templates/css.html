Cвойства
CSS-свойства это определенные параметры оформления, например цвет элемента или текста (color) или цвет фона (background).

#test{

  • transform: scale(1.3); /*трансформирование элемента, scale-изменить размер*/
  • margin: 10px; /*наружние отступы*/
  • margin-top:1px; /*наружние отступы сверху*/
  • margin-left: 30px; /*наружние отступы слево*/
  • margin-right: 30px; /*наружние отступы справо*/
  • padding: 15px; /*внутрение отступы*/
  • color: rgba(0,0,0,0.75); /* цвет текста, 0.75 непрозрачность */
  • border: 2px dashed grey; /*рамка*/
  • background-color: #e6e9f0; /*фон блока*/
  • border-radius: 10px; /*закруглить углы рамки*/
  • opacity: 0.8; /*0.8 непрозрачность блока*/
  • box-shadow: 3px 3px 5px black; /*Тень блока*/
  • width: auto; /*ширина*/
  • height: auto; /*высота*/
  • display: block; /*видимость на экране*/
  • text-align: left; /*расположение текста*/
}
