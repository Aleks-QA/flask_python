Прототип глоссария QA инженера

Глоссарий – словарь основных понятий и терминов, относящихся к определенной сфере деятельности или знания. Главное отличие глоссария от словаря состоит в том, что в глоссарий включаются специализированные термины, связанные с изучением конкретной дисциплины.

Базовая структура html страницы

    <html> является контейнером, который заключает в себе все содержимое веб-страницы.
    • <head> предназначен для хранения служебной информации о странице.
      • <title> Название вкладки </title>
      • <meta теги, через них браузер и поисковые машины считывают информацию о сайте.>
      </head>
    • <body> обозначает границы основного содержимого, которое показывается пользователям.
      • <header> Шапка сайта. Может повторяться на других страницах. <header>
      • <main> Основной контент. Остается только на этой странице. <main>
      • <footer> Подвал сайта. Может повторяться на других страницах. <footer>
      </body>
    </html>

Базовые теги html страницы

  • <p> параграф или текстовый абзац, это блочный элемент, всегда начинается с новой строки.<p>
  • <h1> h1-h6 – семейство заголовочных тегов, показывают относительную важность<h1>
  • <br> устанавливает перевод строки в том месте, где этот тег находится.
  • <a href="https://www.youtube.com/" target="_blank"> Ссылка </a> так выглядит ссылка
  • <strong> Выделенный текст <strong>
  • <em> Курсивный текст <em>
  • <del>Зачеркнутый текст</del>
  • <ins>Подчеркивание текста</ins>
  • <abbr title="Язык разметки гипертекста">HTML</abbr>аббревиатура с атрибутом для полной фразы.
  • <sup> надстрочный <sup> и <sub> подстрочный <sub> текст
  • <input placeholder="поле поиска">
  • <ol> устанавливает нумерованный список, содержит элементы в определенной последовательности.
    <li> 1. элемент списка <li>
    <li> 2. элемент списка <li>
    <li> 3. элемент списка <li>
    </ol>
  • <ul> Используется для неупорядоченного списка - в частности для маркированного списка.
    <li> - элемент списка <li>
    <li> - элемент списка <li>
    <li> - элемент списка <li>
    </ul>
  • <div>разбивает документ на отдельные части, а так же форматирует стиль содержимого.</div>
  • <img scr="https://w7.pngwing.com/... width="100" height="100" > добавить изображение
